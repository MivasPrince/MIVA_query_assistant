# MIVA Query Assistant

The MIVA Query Assistant is a web-based interface designed to enable users to interact with MIVA Open University's BigQuery database using natural language.

---

## Features

- Clean HTML/CSS frontend with JavaScript interactivity
- Accepts natural language queries (e.g., "Show me enrollment by program for 2024")
- Simulates SQL generation and query results using sample data
- Displays results in a responsive data table
- Supports exporting results to CSV and Excel formats

---

## Project Structure

```
.
├── index.html         # Main HTML page with embedded CSS/JS
└── README.txt         # Project documentation (this file)
```

---

## Usage

1. Open `index.html` in your browser
2. Enter a query in natural language
3. Click "Analyze Data"
4. View results, preview SQL, and export if needed

---

## Example Queries

- Show me student enrollment by program for the current academic year  
- What are the top 5 courses with highest completion rates?  
- List all students who failed more than 2 courses in the last semester  
- Show revenue trends by payment method over the last 6 months
